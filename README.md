# vetor_ordenado
Exemplo em Python para ordenação de um vetor

- Facilita busca de elementos
- O(n)
- Pior caso = N

----

Exemplo:

vetor.insere(6) 

vetor.insere(4)

vetor.insere(3)

vetor.insere(5)

vetor.insere(1)

vetor.insere(8)

vetor.imprime()

-----

INDEX - ELEM

0  -  1

1  -  3

2  -  4

3  -  5

4  -  6

5  -  8
